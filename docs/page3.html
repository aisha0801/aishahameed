Project 1

Crash Course Lessons in Airline Disasters

Objective: Analyzed the historical progression of commercial airline crashes (1903-2008) to empower consumers in choosing safer airlines

Key Features: 
   • Dataset: Used a Kaggle dataset for comprehensive analysis.
   • Platforms: Excel, Alteryx, Tableau
   • Interactive Elements: Integrated dynamic filters, buttons, pages, tooltips for an                       engaging user experience.

Dashboard Components:
1.  Time Series Graph: Displays animation of the historical count of commercial airline                crashes over the years.
2.  Geographical Insight: Mapped locations of all crashes, highlighting deadliest airline               attributes and locations with total fatalities.
3.  Ranking of Accident-Prone Airlines: Top 20 list with tooltips providing accident and                 fatality details.
4.  Comparative Analysis: Detailed comparison of the top 5 fatal airlines with interactive             line graphs revealing crash reasons.
5.  Survivability Metrics: Introduced a new metric (survival rate) using Alteryx, displayed via         pie charts for top ten airlines, with corresponding Pareto charts for deeper insights.

Project 3

Alteryx Project: Exploring Causes of Power Outages

Objective: Investigated unconventional causes of power outages.

Methodology:
   • Utilized Alteryx to manipulate and analyze a raw dataset.
   • Sorted animals based on their frequency of causing outages.
   • Converted Alteryx.yxdb into Tableau-accessible .tde format and created visualizations to       depict correlations between animal types and power outages.

Results: Typically, one might attribute these electric mishaps to severe weather or natural disasters, but this dataset focused on a culprit you may have not considered before. If you were wondering animals, you guessed right!

Project 4

UTD Spot Check

As a part of my Systems Analysis and Design class, my team and I created a systems application proposal known as UTD SpotCheck. The objective of this application was to eliminate the hunt for parking that students and faculty face on the UTD campus by allowing them to make reservations for parking spots for an allotted time before arriving on campus. Project 5

Predicting Movie Ratings with Machine Learning

Objective: Developed a predictive model for movie ratings using data science techniques.

Dataset: CodSoft's "Predicting Movie Ratings with Data Science," featuring 5851 movie entries.

Tools Used: Anaconda, JupyterLab, VSCodium, Sklearn, Pandas, Matplotlib, Seaborn.

Project Highlights:
   • Data Preprocessing:
         - Cleaned data using Pandas: filled null values and adjusted data types.
         - Created visualizations with Seaborn and Matplotlib to identify trends and key                          influencers of ratings.
   • Model Development:
        - Employed supervised learning with Linear Regression, Ridge Regression, Lasso                        Regression, and Polynomial Regression models.  
   • Model Evaluation:
        - Assessed models using MAE, MSE, and R-squared metrics for accuracy.
   • Outcome:
        - Built and evaluated multiple regression models to predict movie ratings.
        - Created supplementary visualizations to support findings and provide a                                  comprehensive understanding of data trends and model performance.
